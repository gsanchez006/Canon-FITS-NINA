## What Was Released

### Critical Bug Fix
Fixed last image conversion failure in long sequences where the last images in bursts (30+ shots) would fail to convert due to timestamp-based file mismatching.

### Solution Implemented
- Queue-based sequential processing system (FIFO)
- `ConcurrentQueue<PendingImageData>` for chronological ordering
- `ProcessQueueAsync()` background worker
- Maintained concurrent processing (3 parallel conversions)
- Proper queue drainage on shutdown
